=> Project Overview

This project implements an Image Captioning system using Python. The application generates a simple descriptive caption for an image based on predefined image categories. It demonstrates the basic concept of image caption generation.

=> Features

* Accepts image category input
* Generates image captions
* Simple and user-friendly interface
* Demonstrates AI image captioning concept

=> Technologies Used

* Python 3
