=> Face Detection and Recognition

=> CodSoft Artificial Intelligence Internship - Task 5

=> Project Overview

This project performs face detection and basic face recognition using Python and OpenCV. The application detects faces from a webcam in real time by drawing a bounding box around each detected face.

=> Features

* Real-time face detection
* Webcam support
* Face bounding boxes
* Simple and user-friendly interface

=> Technologies Used

* Python 3
* OpenCV
