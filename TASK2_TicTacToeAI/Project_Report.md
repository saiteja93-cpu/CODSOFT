=> PROJECT REPORT

=> Title

Tic-Tac-Toe AI Using Python

=> Objective

The objective of this project is to develop a Tic-Tac-Toe game where a human player competes against an AI opponent. The project demonstrates the implementation of game logic, user interaction, and basic artificial intelligence concepts.

=> Introduction

Tic-Tac-Toe is a popular two-player game played on a 3x3 grid. In this project, the user plays against the computer (AI). The AI automatically selects moves and competes with the player until a winner is determined or the game ends in a draw.

=> Tools and Technologies Used

* Python 3
* Random Module
* GitHub

=> Features

* Human vs AI Gameplay
* Interactive Command-Line Interface
* Winner Detection
* Draw Detection
* Input Validation
* AI Move Generation

=> Working

1. The game board is initialized.
2. The player enters a position from 1 to 9.
3. The AI selects an available position.
4. The program checks for a winner after each move.
5. The game continues until a player wins or the board is full.

=> Results

The game successfully allows a user to play Tic-Tac-Toe against an AI opponent and correctly determines wins, losses, and draws.

=> Conclusion

This project demonstrates the fundamentals of game development, decision-making logic, and user interaction using Python.
