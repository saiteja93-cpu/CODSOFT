=> Tic-Tac-Toe AI

=> CodSoft Artificial Intelligence Internship - Task 2

=> Project Overview

This project is a Tic-Tac-Toe game developed using Python where a user plays against an AI opponent.

=> Features

* Human vs AI gameplay
* Win detection
* Draw detection
* User-friendly interface
* AI move generation

=> Technologies Used

* Python 3

=> How to Run

1. Install Python 3.
2. Open terminal in the project folder.
3. Run:

python tic_tac_toe.py

=> Sample Gameplay

Player: X

AI: O

Enter position (1-9): 5

AI makes a move.

Continue until a player wins or the match is drawn.
