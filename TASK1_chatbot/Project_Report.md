=> PROJECT REPORT

=> Title

Rule-Based Chatbot Using Python
=> Objective

The objective of this project is to develop a simple chatbot that can interact with users and respond to predefined queries using rule-based logic.

=> Introduction

A chatbot is a software application that simulates conversation with users. This project implements a rule-based chatbot using Python. The chatbot responds to user inputs such as greetings, date, time, help requests, and other predefined questions.

=> Tools and Technologies Used

* Python 3
* Datetime Module
* GitHub

=> Features

* Greeting Responses
* User Name Recognition
* Current Date Information
* Current Time Information
* Help Command
* Internship Information
* Exit Command
* Unknown Query Handling

=> Working

1. The chatbot accepts user input.
2. The input is converted to lowercase.
3. The program checks predefined conditions.
4. A suitable response is displayed.
5. The conversation continues until the user types "bye", "exit", or "quit".

=> Results

The chatbot successfully responds to common user queries and provides a simple conversational experience.

=> Conclusion

This project demonstrates the implementation of a basic rule-based chatbot using Python. It helps in understanding chatbot architecture, user interaction, and conditional programming concepts.
