=> Rule-Based Chatbot

=> CodSoft Artificial Intelligence Internship - Task 1

=> Project Overview

This project is a Rule-Based Chatbot developed using Python. The chatbot interacts with users through predefined rules and responds to common queries such as greetings, date, time, help commands, and more.

The objective of this project is to understand the fundamentals of chatbot development, conversational flow, and basic Natural Language Processing concepts.

---

=> Features

* Greeting Responses
* User Name Recognition
* Current Date Information
* Current Time Information
* Help Command
* Joke Command
* Exit Command
* Unknown Query Handling

---

=>Technologies Used

* Python 3
* Datetime Module
* Random Module
